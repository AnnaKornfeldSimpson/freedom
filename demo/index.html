  • Counter The "Hello World" of FreeDOM, counter shows how to create a basic backend module in FreeDOM and communicate between the page and the module. FreeDOM modules encapsulate the backend of the application and can be migrated across devices with minimal effort. manifest.json counter.js README.md
  • TicTak TicTak demonstrates a single player game making use of the FreeDOM Storage API. TicTak uses the default Storage provider, to store game stats. manifest.json main.js README.md
  • Chat Chat demonstrates how to use the FreeDOM Social API to enable multi-user interactions. In this basic example, the Social API is used to send chat messages, where all users who use the app will join a global buddylist with a GUID. manifest.json main.js README.md
