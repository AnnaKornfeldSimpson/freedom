Available Demos
  • Counter The "Hello World" of FreeDOM, counter shows how a backend module can be used with the FreeDOM library to provide availability across devices and with minimal development effort. manifest.json counter.js README.md
  • TicTak A minimal, complete FreeDOM demo, TicTak demonstrates a single player game making use of a freedom storage service. TicTak shows how to use pluggable services to gain access to privileges beyond those available to standard web applications. manifest.json tictak.js storage.json storage.js README.md
  • Chat A minimal chat demonstration. Anyone using the demo will join a global buddylist with a GUID. The demo makes use of a custom freedom identity provider. Chat shows how one can leverage existing third-party servers to expose an identity and social network. manifest.json main.js identity.json identity.js router.py README.md
